## Overview

FastSkill exposes Claude Code-compatible endpoints under `/v1/*`.

Base URL:

```text
http://localhost:8080/v1
```

<Info>
For local `fastskill serve`, v1 endpoints are unauthenticated.
</Info>

## Endpoints

- `POST /v1/skills`
- `GET /v1/skills`
- `GET /v1/skills/:skill_id`
- `DELETE /v1/skills/:skill_id`
- `POST /v1/skills/:skill_id/versions`
- `GET /v1/skills/:skill_id/versions`
- `GET /v1/skills/:skill_id/versions/:version`
- `DELETE /v1/skills/:skill_id/versions/:version`

## Examples

```bash
# List skills
curl -X GET http://localhost:8080/v1/skills

# Get one skill
curl -X GET http://localhost:8080/v1/skills/my-skill
```

## Error Codes

| Code | Meaning |
|------|---------|
| `BAD_REQUEST` | Invalid request payload or parameters |
| `NOT_FOUND` | Resource not found |
| `INTERNAL_SERVER_ERROR` | Unexpected server error |
