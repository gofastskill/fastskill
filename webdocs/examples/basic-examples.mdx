<Steps>
<Step title="Create a PDF text extractor skill">
```json skills/pdf-extractor.json

  "id": "pdf-text-extractor",

  "skill_file": "pdf-extractor/SKILL.md",
  "enabled": true,
  "execution_timeout": 60,
  "memory_limit_mb": 512,
  "dependencies": [],
  "metadata": {
    "category": "text-processing",
```

```markdown skills/pdf-extractor/SKILL.md
---
name: pdf-text-extractor
description: Extract clean text content from PDF documents
---

# PDF Text Extractor

This skill extracts text from PDF documents.

## Tools

### extract_text_from_pdf

Extracts text content from a PDF document.

**Parameters:**
- `file_path` (string, required): Path to the PDF file
- `page_range` (array, optional): Specific pages to extract (e.g., [1, 2, 3])

**Returns:**
- Extracted text content
- Total pages
- Extracted pages count
```
</Step>

<Step title="Register and use the skill with Rust">
```rust
use fastskill::{FastSkillService, ServiceConfig, SkillDefinition, SkillId};
use std::path::PathBuf;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    // Initialize service
    let config = ServiceConfig {
        skill_storage_path: PathBuf::from("./skills"),
        ..Default::default()
    };

    let mut service = FastSkillService::new(config).await?;
    service.initialize().await?;

    // Create skill definition
    let skill = SkillDefinition::new(
        SkillId::new("pdf-text-extractor".to_string())?,
        "PDF Text Extractor".to_string(),
        "Extract clean text content from PDF documents".to_string(),
        "1.0.0".to_string(),
    );

    // Register skill
    let skill_id = service.skill_manager().register_skill(skill).await?;
    println!("‚úÖ PDF extractor skill registered: {}", skill_id);

    // Discover skills
    let skills = service.metadata_service()
        .discover_skills("extract text from PDF")
        .await?;
    
    println!("Found {} relevant skills", skills.len());
    for skill_meta in skills {
        println!("  - {}: {}", skill_meta.id, skill_meta.name);
    }

    service.shutdown().await?;
    Ok(())
}
```
</Step>
</Steps>

## Integration Examples

### With AI Agents

```rust
use fastskill::{FastSkillService, ServiceConfig};
use std::path::PathBuf;

// Example: Integration with an AI agent using FastSkill
#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let config = ServiceConfig {
        skill_storage_path: PathBuf::from("./skills"),
        ..Default::default()
    };

    let mut service = FastSkillService::new(config).await?;
    service.initialize().await?;

    // Discover relevant skills for a user query
    let user_query = "analyze this text for sentiment";
    let relevant_skills = service.metadata_service()
        .discover_skills(user_query)
        .await?;

    if relevant_skills.is_empty() {
        println!("No skills found for: {}", user_query);
    } else {
        println!("Found {} relevant skills:", relevant_skills.len());
        for skill in relevant_skills {
            println!("  - {}: {}", skill.id, skill.name);
        }
    }

    service.shutdown().await?;
    Ok(())
}
```
